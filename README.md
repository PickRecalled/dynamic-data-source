# dynamic-data-source  动态数据源

### 项目特点
- 支持动态添加数据源
- 添加新数据源后，不用重新启动项目即可使用
- 交流QQ群：17470566
- 本人QQ：913624256
- 如果喜欢，记得star fork 谢谢您的关注



### 使用方法
- 1、localhost:8080/refresh  刷新数据源
- 2、localhost:8080/test     查询数据，传请求头 `dbs = 对应数据源的key`
